About AI Sentia

(sen-tee-uh • Latin for "awareness")

The Problem We're Solving

The AI industry moves at breakneck speed. Every day, there are dozens of new research papers, model releases, funding announcements, and policy changes across 35+ different sources. Keeping up with it all is overwhelming.

Developers need to know about new GitHub repositories and framework updates. Researchers need to track ArXiv papers and academic breakthroughs. Enterprise leaders need policy updates and industry trends. Healthcare professionals need clinical AI validation studies.

AI Sentia exists to solve this information overload problem. We aggregate, filter, and categorize AI news from 35+ global sources, then route it to the right audience using LLM-powered intelligence.

Our Mission

🎯 Make AI news accessible to everyone - No paywalls, no login required, completely free.

🤖 Use AI to fight information overload - LLMs categorize and summarize news so you can scan 100 articles in 5 minutes.

🌍 Connect global AI communities - From Silicon Valley to Bangalore, from healthcare to policy, we bridge ecosystems.

📖 Stay open source - Our code is public on GitHub so anyone can learn, contribute, or build their own aggregator.

How It Works

1

Automated Crawling

Every 24 hours, our GitHub Actions workflow crawls 35+ sources: OpenAI, Anthropic, Google DeepMind, ArXiv, MIT, Stanford, GitHub Trending, YouTube creators, and more.

2

AI-Powered Enrichment

Google Gemini models analyze each article, generate concise summaries, assign personas (Developer, Research, Enterprise, etc.), and extract metadata like industries, companies, and locations.

3

Smart Filtering

We detect trending content using a Hacker News-style algorithm, extract GitHub star counts and YouTube view counts, and deduplicate similar stories from multiple sources.

4

Instant Dashboard

Results are published to a fast, mobile-responsive dashboard with real-time filtering by persona, industry, company, job title, and location. No backend required - everything runs on static JSON.
